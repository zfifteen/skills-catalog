# Observable Plot

## What Problem This Solves

This reference covers the fastest grammar-first path for many JavaScript charts.

## When to Use It

Use this when a JavaScript or TypeScript workflow needs quick, readable charts with layered marks, transforms, and a compact API.

## Key Takeaways

- Plot is excellent for exploratory and explanatory tabular charts with concise code.
- It benefits from grammar-of-graphics thinking without requiring large hand-authored specs.
- It is often the right default before reaching for D3.

## Common Mistakes

- Expecting Plot to replace every bespoke interaction or custom layout need.
- Ignoring the point where a Plot composition has become harder to reason about than a simpler tool or a custom chart.
