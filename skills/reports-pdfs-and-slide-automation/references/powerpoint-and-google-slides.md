# PowerPoint and Google Slides

## What Problem This Solves

This reference covers slide automation workflows for chart-heavy presentations.

## When to Use It

Use this when generating or updating decks programmatically.

## Key Takeaways

- Generate figure assets first, then place them into slide geometry.
- `pptxgenjs` is a strong PowerPoint automation path for JavaScript and TypeScript workflows.
- Google Slides works best through explicit page and object placement via the API.

## Common Mistakes

- Letting charts determine slide layout implicitly.
- Ignoring theme, title, and caption consistency across slides.
