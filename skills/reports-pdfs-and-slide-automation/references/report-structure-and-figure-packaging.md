# Report Structure and Figure Packaging

## What Problem This Solves

This reference covers how to turn charts into report-ready deliverables rather than isolated graphics.

## When to Use It

Use this when preparing report sections, asset slots, figure sizes, captions, and appendices.

## Key Takeaways

- Lead with the decision or claim, not the chart inventory.
- Package figures as durable assets with deterministic dimensions.
- Align labels, annotation density, and typography to the final slot size.

## Common Mistakes

- Designing charts at arbitrary sizes and hoping the document will absorb them.
- Treating the narrative text and the visual as separate artifacts.
