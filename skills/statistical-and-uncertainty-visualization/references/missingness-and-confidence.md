# Missingness and Confidence

## What Problem This Solves

This reference covers the parts of analytical truth that often vanish from final charts.

## When to Use It

Use this when incomplete data or confidence levels materially affect interpretation.

## Key Takeaways

- Missingness deserves explicit visual treatment when it influences trust or comparability.
- Confidence, certainty, and support should be described in language the audience can interpret.
- Sometimes the right move is a table or annotation rather than a new mark type.

## Common Mistakes

- Encoding missing values as zero.
- Treating confidence as self-explanatory.
