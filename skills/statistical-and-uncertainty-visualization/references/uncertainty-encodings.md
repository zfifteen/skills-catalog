# Uncertainty Encodings

## What Problem This Solves

This reference helps choose how to show uncertainty rather than omit it.

## When to Use It

Use this when a claim depends on estimation, interval width, simulation, model uncertainty, or sampling variability.

## Key Takeaways

- Error bars, error bands, intervals, and ensemble views each communicate different kinds of uncertainty.
- Explain what the uncertainty measure actually is.
- Pair uncertainty with context so viewers know how much it changes the conclusion.

## Common Mistakes

- Showing intervals without defining them.
- Using uncertainty as a decorative band instead of a meaningful claim.
